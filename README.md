---

## ✨ 增强功能

| 功能 | 说明 |
|------|------|
| 🎨 **Mermaid 图表渲染** | 聊天中渲染流程图/时序图，支持拖动、滚轮缩放、双击重置 |
| 🌐 **HTML 实时预览** | HTML 代码块直接预览，支持预览/源码切换 |
| 📥 **保存到本地** | SVG/HTML 一键保存，弹出系统对话框选择路径 |
| 📂 **TUI 文件浏览器** | 终端侧栏文件树，支持文件内容预览 |
